AgriVision API Documentation

Aplikasi Petani Cerdas untuk DeteksiPenyakit Tanaman dan Edukasi Pertanian

Create Education

Endpoint: POST /educations

Description: Menambahkan edukasi penyakit tanaman baru.

Request Payload (JSON):


{
  "diseaseName": "string",
  "characteristics": "string",
  "symptoms": "string",
  "cause": "string",
  "prevention": ["string", "string", ...],
  "treatment": {
    "biological": "string",
    "chemical": "string"
  },
  "videoEmbed": "string"
}

Responses:

  • 201 Created: Berhasil menambahkan edukasi.
    { "status": "success", "message": "Edukasi penyakit tanaman berhasil ditambahkan", "data": { "educationId": "string" } }
  • 400 Bad Request: Jika ada field kosong atau format salah.
    { "status": "fail", "message": "Gagal menambahkan edukasi ..." }

Get All Educations

Endpoint: GET /educations

Description: Mengambil semua data edukasi penyakit tanaman.

Responses:

  • 200 OK:
    { "status": "success", "data": { "educations": [ { ... }, { ... } ] } }

Get Education by ID

Endpoint: GET /educations/{educationId}

Description: Mengambil detail edukasi berdasarkan ID.

Path Parameters:

  • educationId (string) - ID unik dari edukasi.

Responses:

  • 200 OK:
    { "status": "success", "data": { "education": { ... } } }
  • 404 Not Found: Jika ID tidak ditemukan.
    { "status": "fail", "message": "Edukasi penyakit tanaman tidak ditemukan, Id tidak ditemukan" }

Update Education by ID

Endpoint: PUT /educations/{educationId}

Description: Memperbarui data edukasi berdasarkan ID.

Path Parameters:

  • educationId (string) - ID unik dari edukasi.

Request Payload (JSON):


{
  "diseaseName": "string",
  "characteristics": "string",
  "symptoms": "string",
  "cause": "string",
  "prevention": ["string", "string", ...],
  "treatment": {
    "biological": "string",
    "chemical": "string"
  },
  "videoEmbed": "string"
}

Responses:

  • 200 OK:
    { "status": "success", "message": "Edukasi penyakit tanaman berhasil diperbarui" }
  • 400 Bad Request: Jika ada field kosong.
    { "status": "fail", "message": "Gagal memperbarui edukasi penyakit tanaman. Mohon isi semua field." }
  • 404 Not Found: Jika ID tidak ditemukan.
    { "status": "fail", "message": "Gagal memperbarui edukasi penyakit tanaman. Id tidak ditemukan" }

Delete Education by ID

Endpoint: DELETE /educations/{educationId}

Description: Menghapus data edukasi berdasarkan ID.

Path Parameters:

  • educationId (string) - ID unik dari edukasi.

Responses:

  • 200 OK:
    { "status": "success", "message": "Edukasi penyakit tanaman berhasil dihapus" }
  • 404 Not Found: Jika ID tidak ditemukan.
    { "status": "fail", "message": "Gagal menghapus edukasi penyakit tanaman. Id tidak ditemukan" }

Detection Page

Endpoint: GET /detection

Description: Menampilkan halaman deteksi penyakit tanaman.

Responses:

  • 200 OK:
    Halaman deteksi penyakit tanaman

404 Not Found

Responses:

  • 404 Not Found:
